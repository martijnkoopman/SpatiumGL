Create orthographic projection matrix

Project from orthographic view volume to canonical view volume.

Beware: near and far should be negative and the following is true: 0 > near > far far is a negative number of larger absolute value than near.

[in]aspectAspect ratio (w/h)

[in]farDistance to far clipping plane (Z coordinate)
